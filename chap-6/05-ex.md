### Resolvez la relation de récurrence :

$$C_{N} = C_{N/2} + N^2$$
  
$$N \ge 2, C_1 = 0$$

Poursuivons la récurrence :

$$C_{N} = C_{N/2} + N^2$$

$$C_{N/2} = C_{N/4} + (N/2)^2$$

$$C_{N/4} = C_{N/8} + (N/4)^2$$

$$...$$

$$C_{N} = C_{N/2} + N^2\times(\frac{1}{4})^0$$

$$C_{N/2} = C_{N/4} + N^2\times(\frac{1}{4})^1$$

$$C_{N/4} = C_{N/8} + N^2\times(\frac{1}{4})^2$$

$$C_{N/8} = C_{N/16} + N^2\times(\frac{1}{4})^3$$

$$...$$

Donc,

$$C_{N} = C_1 + N^2 ((\frac{1}{4})^0 + (\frac{1}{4})^1 + (\frac{1}{4})^3 + ...+(\frac{1}{4})^n)$$

or:

$$(\frac{1}{4})^0 + (\frac{1}{4})^1 + (\frac{1}{4})^3 + ...+(\frac{1}{4})^n = \dfrac{(\frac{1}{4})^{n+1} - 1}{\frac{1}{4} -1}$$

Pour n tendant vers l'infini, $$(\frac{1}{4})^{n+1} = 0$$:

Donc $$(\frac{1}{4})^0 + (\frac{1}{4})^1 + (\frac{1}{4})^3 + ...+(\frac{1}{4})^n\text{ tend vers }\dfrac{4}{3}$$

Ainsi, $$C_{N}$$ est de l'ordre de $$\dfrac{4}{3} \times N^2$$
